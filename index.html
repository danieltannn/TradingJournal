GitHub token

A Personal Access Token lets the app save your data directly to your GitHub repo. Stored only in this browser.

  1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Click Generate new token (classic)
  3. Give it a name, tick the repo scope, click Generate
  4. Copy and paste it below
